# user_env

새로운 PC 및 서버에서 빠르게 사용자 환경을 구성할 수 있도록 도와주는 도구

## 지원되는 배포판

* MacBook(검토 필요)
* CentOS(검토 필요)
* Ubuntu(final)

## 통합 환경 구성

다음을 실행하면 zsh 환경이 구성 됨
* 모든 배포판에서 동일하며, 현재 ubuntu만 검증했음

```bash
bash set_env.sh
```
